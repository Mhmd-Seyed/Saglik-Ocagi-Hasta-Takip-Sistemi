# 🏥 Hospital Patient Tracking System (Sağlık Ocağı Hasta Takip Sistemi)

A robust desktop application designed to streamline patient registration, appointment scheduling, and medical record management for healthcare clinics. Developed using **C#** and **SQL Server**.

## 🚀 Key Features
- **Patient Management:** Full CRUD operations for patient records.
- **Dynamic Tracking:** Real-time monitoring of patient status and history.
- **Secure Database:** Integrated with SQL Server for reliable data storage and complex queries.
- **User-Friendly Interface:** Intuitive UI built with Windows Forms (WinForms).

## 🛠 Tech Stack
- **Language:** C# (.NET Framework)
- **Database:** Microsoft SQL Server
- **Architecture:** Tiered Architecture (Logic, Data, UI)

## ⚙️ How to Run
1. Clone the repository.
2. Open the solution file `.sln` in **Visual Studio**.
3. Execute the SQL script provided in the `/Database` folder to set up your tables and stored procedures.
4. Update the connection string in `App.config`.
5. Build and Run!
